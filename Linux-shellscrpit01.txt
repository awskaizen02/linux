Packet manager is the tool that install , update and removes software packages while also handling dependencies automaticall.It's one of the most essiential components of any linux distribution.

Installs software from repositories (official or third-party).

Updates packages to newer versions.

Removes packages cleanly without leaving broken dependencies.

Resolves dependencies (ensures required libraries are installed).

Verifies versions and manages security patches.



Distribution Family	Package Manager					Notes
Debian/Ubuntu		APT (Advanced Package Tool) with dpkg	Uses .deb packages. Commands: apt install, apt update.

RedHat/CentOS/Fedora	DNF (Dandified Yum) / YUM		Uses .rpm packages. Commands: dnf install, yum update.

Arch Linux		pacman					Lightweight, fast. Command: pacman -S package.

openSUSE		zypper					RPM-based, powerful dependency resolution.

Gentoo			Portage					Source-based, highly customizable.

Cross-distro 		Snap, Flatpak, AppImage			Universal formats, sandboxed apps, 
(containerized)							portable across distros.


NixOS			Nix					Declarative, reproducible builds.

#!/bin/bash

load=$(uptime | awk '{print $(NF -2)}' | sed 's/,//')

if (( $(echo "$load < 1.0" | bc -l) )); then
        echo "system load is Low"
elif (( $(echo "$load < 3.0" | bc -l) )); then
        echo "system load is Medium"
else
        echo "system load is High"
fi
